# README

### 1. Clone the project
```
git clone https://github.com/maktezi/POS.git
```

### 2. Goto project directory
```
cd POS
```

### 3. Copy the .env.example to .env then Install Composer
```
.env.example -> .env

&

composer install
```
### 4. Run migrations and db seed
```
php artisan migrate:fresh
php artisan db:seed
```

### 5. Then install npm
```
npm install
```

### 6. Then run the server
```
 npm serve
```
