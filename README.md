# README

### 1. Clone the project
`
git clone https://github.com/maktezi/POS.git
`

### 2. Goto project directory
`
cd POS
`

### 3. Install Composer
`
composer install
`

### 4. Run migrations
`
php artisan migrate
`

### 5. Goto frontend directory then install npm or bun
`
npm install
`
or
`
bun install
`

### 6. Go back to the POS directory
`
cd ..
`

### 7. Then run the server
`
 npm dev
`
or
`
bun dev
`
